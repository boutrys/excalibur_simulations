Excalibur simulation Licensing
============================================

This file attempts to include all licenses that apply within Excalibur simulation (this program) source tree, in particular any that are supposed to be exposed to the end user for credit requirements for instance. 

Excalibur simulation allows to performe numerical simulations in order to assess type I error and power of aggregation test and to construct an ensemble method made of the optimal selection of aggregation test being compared.
